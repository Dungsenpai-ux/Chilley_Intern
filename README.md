## Backend
- Framework: Gin
- Storage: In-memory (resets on restart)
- APIs: POST/GET/PUT/DELETE /tasks

### Run Locally
```bash
go run main.go
```

## Frontend
- Framework: React
- API Calls: Axios

### Run Locally
```bash
npm install 
```
```bash
npm start
```
## Usage
- Add tasks.
- Check/uncheck to complete.
- Delete tasks.

Note: In-memory storage, no persistence.
Frontend: https://github.com/Dungsenpai-ux/UI_Chilley
